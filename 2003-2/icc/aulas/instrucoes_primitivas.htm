Instruções Primitivas

Introdução

  • São aquelas que efetuam tarefas essenciais na execução de algoritmos
    • Entrada de Dados
    • Saída de Dados
    • Atribuição

Atribuição

  • Variável <- Expressão;
  • O valor resultante da expressão é armazenado na variável
  • Importante atentar para a necessidade de garantir que a variável tenha tipo compatível com o resultado da expressão.
  • Exemplo:
variáveis
   Numero : inteiro;
   Preco : real;
   Nome : literal;
início
   Numero <- 42;
   Preco <- 4.5;
   Nome <- 'Maria';
   Numero <- Numero * Preco; { errado }
   Preco <- Preco * Numero;
   Preco <- Numero;
   Nome <- Preco; { errado }
   Numero <- Numero / 3; { errado }
fim.

Saída de Informações

  • Promove a exibição, para o usuário, das informações
  • escreva( lista );
    • Onde lista consiste numa relação de variáveis constantes ou expressões.
  •  Exemplo:
variáveis
   Produto : literal[20];
   Quantidade : inteiro;
   PrecoUnitario : real;
início
   Produto <- 'arroz';
   Quantidade <- 20;
   PrecoUnitario <- 0.75;
   escreva( 'PRODUTO: ',  Produto, ' QUANTIDADE: ', Quantidade,
            'PREÇO: ', Preco, ' PREÇO UNITÁRIO: ', PrecoUnitario );

Entrada de Dados

  • Possibilita a leitura dos dados a serem manipulados pelos programas
  • leia( lista );
    • Onde lista consiste numa relação de variáveis
  • Exemplo:
variáveis
   Produto : literal[20];
   Quant : inteiro;
   Preco : real;
início
   leia( Produto, Quant, Preco );

Amigabilidade

  • Propriedade dos algoritmos relativa à boa interação com os usuários.
  • Para garantir mínima amigabilidade os algoritmos devem apresentar:
    • Mensagens de orientação
    • Exibir títulos e subtítulos
    • Som (beeps) e cores adequadas
    • Antes da leitura dos dados, exibição de mensagem (literal) solicitando-os. Exemplo:

escreva ( 'Preço unitário: ' );
leia ( PrecoUnitario );

  • Exemplo
algoritmo Calculadora;
variávies
   Operando,
   Resultado : real;
   Operacao : literal[1];
início
   escreva ('Calculadora');
   escreva ('Selecione a operação: +, -, *, /');
   leia (Operacao);
   escreva ('Digite o primeiro operando: ');
   leia (Resultado);
   escreva ('Digite o segundo operando: ');
   leia (Operando);
   se Operacao = '+' então
      Resultado <- Resultado + Operando
   senão
      se Operacao = '-' então
         Resultado <- Resultado - Operando
      senão
         se Operacao = '*' então
            Resultado <- Resultado * Operando
         senão
            se Operacao = '/' então
               Resultado <- Resultado / Operando;
   escreva ('O resultado é: ', Resultado);
fim.
