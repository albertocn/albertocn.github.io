Turbo Pascal: Instruções de Uso

Introdução

  • O Turbo Pascal 7 é um ambiente integrado de desenvolvimento (IDE), que permite editar, compilar e executar programas escritos na linguagem Pascal.
  • A compilação é o processo pelo qual o programa escrito em linguagem de alto nível (Pascal) é convertido em um programa executável (EXE) que contém instruções apropriadas para o computador.
  • Existem outras IDE's e compiladores para o Pascal, mas esse será adotado no nosso curso.
  • Em um ambiente integrado de desenvolvimento é mais fácil programar, pois podemos editar, compilar e executar programas em um ambiente que proporciona um conjunto de facilidades.

Instalação

  • Criar um diretório chamado TP7 no diretório raiz. (C:\TP7)
    • Se tiver baixado o arquivo compactado TP7.zip, descompacte e coloque os arquivos (TURBO.EXE, (2) TURBO.ICO, (3) TURBO.TPH e (4) TURBO.TPL) no diretório C:\TP7.
    • Caso contrário, copie os arquivos (1) TURBO.EXE, (2) TURBO.ICO, (3) TURBO.TPH e (4) TURBO.TPL para o diretório C:\TP7.
  • O Turbo Pascal 7 é um ambiente integrado de desenvolvimento (IDE), que permite editar, compilar e executar programas escritos na linguagem Pascal.
  • Para facilitar o uso do TP7, pode-se criar um atalho para o arquivo TURBO.EXE e colocá-lo na área de trabalho, menu ou barra de tarefas.

Principais Comandos

  • Menu File
    • New => Cria um novo arquivo PAS. Deve-se informar o nome e diretório para o arquivo.
    • Open => Abre um arquivo PAS existente. Deve-se informar o nome e diretório do arquivo.
    • Save => Salva o arquivo que está sendo editado.
    • Save as => Salva o arquivo que está sendo editado com um nome/diretório diferente.
    • Save all => Salva todos os arquivos que estão sendo editados.
    • Change dir => Muda o diretório corrente. O padrão é o diretório de instalação do TP7.
    • Print => Imprime o arquivo sendo editado.
    • Printer setup => Configuração de impressão.
    • Dos shell => Executa o prompt do MS-DOS. Para voltar ao TP7, execute EXIT.
    • Exit => Sai do TP7.
  • Menu Edit
    • Undo => Desfaz um comando.
    • Redo => Refaz um comando que foi desfeito.
    • Cut => Corta a parte do programa selecionado, colocando-a na área de transferência.
    • Copy => Copia a parte do programa selecionado, colocando-a na área de transferência.
    • Paste => Cola o conteúdo da área de transferência.
    • Clear => Limpa a parte do programa selecionado.
    • Show clipboard => Exibe o conteúdo da área de transferência
  • Menu Search
    • Find => Busca por um conjunto de caracteres qualquer no programa.
    • Replace => Substitui um conjunto de caracteres no programa por outro.
    • Search again => Executa a busca até a próxima ocorrência do conjunto de caracteres no programa.
    • Go to line number => Move o cursor para a linha especificada.
  • Menu Run
    • Run => Executa o programa corrente. Após a execução, volta ao editor.
  • Menu Compile
    • Compile => Compila o arquivo atual
    • Make => Compila o arquivo atual e todos os dependentes modificados
    • Build => Compila o arquivo atual e todos os dependentes (mesmo sem terem sido modificados)
    • Destination => Pode indicar Memory ou Disk. O último grava um arquivo (EXE) que pode ser executado fora do ambiente do TP7.
  • Menu Options
    • Open => Abre um arquivo de configuração da IDE.
    • Save => Salva a configuração do IDE.
    • Save as => Salva a configuração do IDE, dando-lhe um nome e diretório.
  • Menu Window
    • Tile => Divide a tela em janelas para permitir visualização de mais de um programa ao mesmo tempo.
    • Cascade => Arruma as janelas em cascata, permitindo ver apenas o nome do programa e o conteúdo do que que está na janela da frente.
    • Close all => Fecha todas as janelas.
    • Size/Move => Permite dimensionar e mover janelas.
    • Zoom => Faz com que a janela atual ocupe toda a tela disponível.
    • Next => Passa da janela atual para a seguinte.
    • Previous => Volta para a janela anterior.
    • Close => Fecha a janela atual.
    • List => Exibe uma lista das janelas abertas (cada uma com um programa).
  • Menu Help
    • Contents => Mostra o conteúdo da ajuda.
    • Index => Permite procurar comandos através do nome.
