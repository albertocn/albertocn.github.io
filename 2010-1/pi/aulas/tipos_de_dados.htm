Tipos de Dados

Introdução

São elementos de composição de um algoritmo:

  • Instruções - Ações
  • Dados - Entrada
  • Informações - Saída

Os dados classificam-se conforme os tipos de seus conteúdos (significados, interpretações) compondo os tipos de dados.

Tipos de Dados

Numéricos

  • Compostos por números
  • Podem ser:
    • Inteiros -  positivos ou negativos e não apresentam componentes decimais. Exemplo: idade
    • Reais - positivos ou negativos e podem ter componentes fracionários. Exemplo: preço

Literais

  • Composto por letras, dígitos numéricos e/ou símbolos especiais
  • Também denominados alfanuméricos
  • São assinalados por aspas ou apóstrofos
  • O comprimento N de um literal L consiste no número N de caracteres apresentados por L
  • Exemplo: 'casa' (comprimento 4), 'manaus' (comprimento 6)

Lógicos

  • Podem assumir os valores: verdadeiro ou falso
  • São denominados também booleanos
  • Exemplo: dado que controla exibição da saída
se Resposta = verdadeiro então
  escreva ('Afirmativo')
senão
  escreva ('Negativo');

Exemplo

algoritmo ConvênioMédico;
variáveis
  Idade: inteiro;
  Plano: literal[8];
  Base,
  Preço : real;
  PlanoValido: lógico;
início
  leia(Plano);
  PlanoValido <- verdadeiro;
  se Plano = 'B' então
    Base <- 15.00
  senão
    se Plano = 'A' então
      Base <- 20.00
    senão
      se Plano = 'Especial' então
        Base <- 25.00
      senão
        inicio
          escreva ('Plano inexistente');
          PlanoValido <- falso;
        fim
  se PlanoValido = verdadeiro então
    inicio
      leia(Idade);
      se Idade <= 20 então
        Preço <- Base * 3
      senão
        se Idade >= 21 e Idade <= 40 então
          Preço <- Base * 2
        senão
          se Idade >= 41 então
            Preço <- Base * 2.5;
      exibir (Preço);
    fim
fim.
