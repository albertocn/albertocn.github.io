Variáveis

Introdução

Exemplo

  • Algoritmo que calcula as possibilidades de empréstimo bancário de um dado salário:
algoritmo CalculaEmprestimo;
variáveis
   Salario,
   Emprestimo : real
início
   leia(Salario);
   se Salario >= 175.00 então
      início
         Emprestimo <- 150.00;
         enquanto 0.12 * Emprestimo < 0.15 * Salario faça
            início
               escreva('EMPRÉSTIMO:', Emprestimo);
               escreva('PRESTAÇÃO:',  0.12 * Emprestimo);
               Emprestimo <- Emprestimo + 50.00;
            fim
      fim;
   senão
      escreva('SALARIO INSUFICIENTE');
fim.
 

Rastreando

Salário - 300.00 (comprometimento mensal = 45)
Empréstimo 150  200  250  300  350
Prestação   18   24   30   36   42

Conceito

  • São elementos computacionais cujos conteúdos variam durante a execução de um programa/algoritmo

Características

  • Referem-se a espaços em memória e funcionam como gavetas que armazenam dados ou informações.
  • Ao armazenar-se um conteúdo X numa variável que contém Y, X substitui Y e Y é irrecuperável.
  • São referenciados nos programas pelos seus nomes ou identificadores.
  • Podem ser numéricas, alfanuméricas ou lógicas.

Declaração

  • A utilização de variáveis em algoritmos requer a declaração prévia dessas
  • É efetuada na seção variáveis, como segue:
    variáveis
   Identificador : Tipo;
  • onde:
    • Tipo - real, inteiro, literal ou logico; conforme os dados contidos nas variáveis
    • Identificador - nome qualquer, preferencialmente significativo usado para referenciar as variáveis. Devem ser únicos.

Exemplo

variáveis
   Preço:  real;
   Idade:  inteiro;
   Nome:   literal[30];
   Valido: logico;

Variáveis Literais

  • Usadas para armazenar/manipular dados alfanuméricos
  • Declaração:
variáveis
   Identificador: literal[N];
  • onde:
    • N - indica o nº máximo de caracteres componentes da variável. Também denominado tamanho.
  • O tamanho máximo que uma variável literal pode assumir é 255.
  • É possível fazer referência a cada dígito componente de uma variável literal.
variáveis
   Nome: literal[30];
   Letra: literal[1];
início
   Nome <- 'Marta';
   Letra <- Nome[3];
  • É possível concatenar o conteúdo de variáveis literais.
variáveis
   Nome : literal[10];
   Sobrenome: literal[10];
   NomeCompleto: literal[21];
início
   Nome <- 'Mario';
   Sobrenome <- 'Andrade'
   NomeCompleto <- Nome + ' ' + Sobrenome
  • A variável NomeCompleto conterá 'Mario Andrade'.

Variáveis Lógicas

  • Possuem apenas dois valores possíveis: verdadeiro ou falso.
  • São usadas geralmente em instruções de controle.
  • Declaração
variáveis
   Identificador : Logica;
  • Exemplo
variáveis
   Autorizacao : Logica;

Variáveis Numéricas

  • Conforme os dados que armazenam, podem ser do tipo inteiro ou real.
  • Declaração:
variáveis
   Identificador1 : inteiro;
   Identificador2 : real;
  • Exemplo
variáveis
   Idade: inteiro;
   Preço: real;
